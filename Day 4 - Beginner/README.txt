Head or Tails - программа бросат монетку, где с помощью модуля *random
всегда выпадает случайное из двух значений: Орел или Решка.

Banker Roulette - этой программой мы, с помощью модуля *random,
выбираем из списка гостей (список пишется через запятую) того, 
кто будет оплачивать чек за весь обед.

Treasure Map - тут мы написали код, который отмечает квадрат c сокровищами
на карте map знаком X, используя двузначную систему. 
Первая цифра — это номер вертикального столбца,
а вторая цифра — номер горизонтальной строки.

Rock, Paper, Scissors - это игра в Камень, Ножницы, Бумага, где игрок
и компьютер соревнуются друг с другом, все согласно всемирным правилам
с добавленными иллюстрациями от ARTSCII для Python. Также используется модуль
*random для случайного выбора компьютера.
